Logs should (almost) always contain contextualized information. String literals can not contain contextualized information and are therefore usually not useful log output.

Non-compliant Code Example

private void programFailed(Program program) {
    LOGGER.info("A program failed!");
}

Compliant Solution

private void programFailed(Program program) {
    LOGGER.info("The program {} failed!", program.getName());
}
